# 📚 Social Learning App

A modern **Social Learning App** built with Android (Jetpack Compose, Firebase).  
The app allows users to connect, chat, and learn through interactive quizzes and social engagement.

---

## ✨ Features

- 🔑 **Firebase Authentication** – Secure login & registration with email/password.  
- 💬 **Real-time Chat** – Send and receive messages instantly with delivery status (✓ sent, ✓✓ seen).  
- 👤 **User Presence** – Online/offline status indicators.  
- 📚 **Quizzes Module** – Interactive quizzes for learning.  
- 🎨 **Custom UI** – Clean design with Compose and branded logo.  
- 🔔 **Push Notifications** – Instant updates using FCM.
